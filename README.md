SolarSystemVisualizer
=====================

This project displays a 3D scene of a solar system where a small spaceship can be launched on a trajectory set by the user.<br/>
This project was made with libQGLViewer and CGLA (see documentation in CGLA directory).<br/>
Made as a school project with Olivier Champeau for the LOG750 class (Infography/3D) at ÉTS university in 2013
